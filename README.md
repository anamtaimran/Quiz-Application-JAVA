# Quiz Application 🎯

## About
A GUI based Quiz Application built in Java using
Java Swing as a Semester 2 OOP group project.

## Features
- Graphical user interface built with Java Swing
- Multiple choice questions with clickable buttons
- Live countdown timer for each question
- Real time score tracking
- Final results screen showing correct and wrong answers
- Questions hardcoded and organized using OOP structure

## Language
Java

## Technologies and Concepts Used
- Java Swing — GUI framework
- ActionListener — button click events
- Timer — countdown functionality
- Classes and Objects
- Inheritance
- Encapsulation
- Constructors and Methods
- Arrays — storing questions and answers
- Conditional logic — checking correct answers


## What I Learned
This project was my first experience building a
real graphical application. It taught me how OOP
concepts connect directly to user interfaces,
how event handling works in Java Swing, and
how to implement a timer in a real application.
